## Overview

NoteX transforms your meetings by providing real-time AI transcription, intelligent summarization, and automatic organization of action items. You capture every detail without manual note-taking, freeing you to focus on discussions. Built by Sotatek, NoteX integrates seamlessly with popular platforms like Zoom, Google Meet, and Microsoft Teams.

For developers, integrate via API:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/transcribe/start', {
    method: 'POST',
    headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` },
    body: JSON.stringify({ meetingId: 'meeting-123' })
  });
  const data = await response.json();
  console.log(data.transcriptUrl);
  ```
  ```python
  import requests
  response = requests.post(
      'https://api.example.com/v1/transcribe/start',
      headers={'Authorization': f'Bearer {YOUR_API_KEY}'},
      json={'meetingId': 'meeting-123'}
  )
  print(response.json()['transcriptUrl'])
  ```
</CodeGroup>
